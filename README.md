# Three.JS Experiments 

Some toy programs made with Three.JS

## Rainbow Cubes

Moving cubes plus movement:

* WASD for direction
* Q for "down"
* E for "up"
* Click + drag mouse to change view angle

To view:

    cd rainbow-cubes
    python3 -m http.server

view [localhost:8000](http://localhost:8000) in your browser
